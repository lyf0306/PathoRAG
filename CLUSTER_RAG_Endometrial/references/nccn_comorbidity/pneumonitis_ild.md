# NCCN 用药安全指南：间质性肺病/肺炎 (Pneumonitis / ILD)

## 风险等级
- **严重性**: 极高
- **关注药物**: PD-1/PD-L1 抑制剂、依维莫司（mTOR 抑制剂）、博来霉素、吉西他滨

## 临床警示
- **免疫检查点抑制剂**：免疫相关性肺炎是最严重的 irAEs 之一，致死率高
- **依维莫司**：mTOR 抑制剂相关性非感染性肺炎（发生率约 15%）
- 基线存在 ILD 的患者风险显著增加

## 管理标准
1. **基线评估**：
   - 治疗前胸部 CT 检查
   - 排除已存在的 ILD
   - 肺功能检查（DLCO 评估弥散功能）
2. **活动性 ILD 患者**：
   - **绝对禁忌或极度慎用** PD-1 抑制剂及依维莫司
3. **治疗中监测**：
   - 每周期前询问有无新发呼吸困难、干咳、低氧
   - 可疑症状时立即行胸部 CT + 肺功能检查
4. **管理方案**：
   - 1级（轻度）：暂停用药，监测，可口服激素
   - ≥2级（中重度）：永久停用，高剂量全身激素（甲泼尼龙 1-4 mg/kg/d）
   - 激素抵抗型：加用英夫利西单抗或吗替麦考酚酯
