# NCCN 用药安全指南：高血压 (Hypertension)

## 风险等级
- **严重性**: 高
- **关注药物**: 仑伐替尼、贝伐珠单抗、其他抗血管生成药物

## 临床警示
抗血管生成药物（VEGF/VEGFR 抑制剂）极易诱发高血压危象、脑卒中或严重蛋白尿。

## 管理标准
1. **用药前**：血压必须控制在 **130/80 mmHg** 以下
2. **治疗期间**：
   - 发生 ≥3级高血压 → 暂停用药 + 强化降压治疗
   - 发生 4级高血压 → 永久停药
3. **监测频率**：每周监测血压，稳定后可每2周

## 药物相互作用
- 贝伐珠单抗与绝大多数降压药无显著相互作用
- 仑伐替尼：已知可导致治疗相关性高血压（发生率 >40%）
- 紫杉类：预处理地塞米松可能引起一过性血压升高
