# NCCN 用药安全指南：周围神经病变 (Peripheral Neuropathy)

## 风险等级
- **严重性**: 中高
- **关注药物**: 紫杉醇、多西他赛、顺铂、卡铂、奥沙利铂、长春新碱

## 临床警示
紫杉类和铂类药物是导致化疗相关性周围神经毒性（CIPN）最常见的原因。基线 ≥2级神经病变者发生致残性 CIPN 的风险极高。

## 管理标准
1. **基线评估**：
   - 用药前进行神经功能评估（感觉、运动、深腱反射）
   - 已有 ≥2级神经病变者，权衡风险与获益
2. **剂量调整**：
   - 发生 ≥2级 CIPN → 暂停用药直至恢复至 ≤1级
   - 恢复后减量 20-25% 重新开始
   - ≥3级 CIPN → 停止该药物
3. **替代方案**：
   - 紫杉醇 → 多西他赛（交叉毒性不完全，可能减轻）
   - 紫杉醇 → 白蛋白结合型紫杉醇（神经毒性相似但有不同药代动力学）
   - 顺铂 → 卡铂（神经毒性较低）
4. **对症治疗**：
   - 疼痛性神经病变：度洛西汀（唯一有证据支持的药物）
   - 补充治疗：加巴喷丁、利多卡因贴剂
