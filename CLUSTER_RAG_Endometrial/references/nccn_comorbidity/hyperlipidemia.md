# NCCN 用药安全指南：高脂血症/脂肪肝 (Hyperlipidemia)

## 风险等级
- **严重性**: 中低
- **关注药物**: 依维莫司（mTOR 抑制剂）、芳香化酶抑制剂（来曲唑、阿那曲唑等）

## 临床警示
- mTOR 抑制剂（依维莫司）可导致血脂升高（总胆固醇和甘油三酯），发生率约 50-70%
- 芳香化酶抑制剂可能轻度升高 LDL-C
- 脂肪肝本身通常不直接影响化疗药物选择，但可能提示代谢综合征

## 管理标准
1. **基线评估**：
   - 治疗前检测空腹血脂（TC、TG、LDL-C、HDL-C）
2. **使用依维莫司时**：
   - 治疗开始前评估血脂
   - 治疗开始后每 2-4 周监测
   - 根据 NCEP-ATP III 指南启动他汀类药物治疗
3. **处理方案**：
   - 高胆固醇血症：他汀类药物（阿托伐他汀、瑞舒伐他汀）
   - 高甘油三酯血症：贝特类药物（非诺贝特）
4. **通常无需因单纯高脂血症调整化疗方案**
