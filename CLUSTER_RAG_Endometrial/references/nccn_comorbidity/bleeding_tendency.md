# NCCN 用药安全指南：出血倾向/活动性出血 (Bleeding Tendency)

## 风险等级
- **严重性**: 极高
- **关注药物**: 贝伐珠单抗、仑伐替尼、其他抗血管生成药物

## 临床警示
抗血管生成药物（贝伐珠单抗、仑伐替尼等）通过抑制 VEGF 通路影响血管完整性，可引发致命性大出血。

## 管理标准
1. **绝对禁忌**：活动性出血患者使用贝伐珠单抗或仑伐替尼
2. **用药前评估**：
   - 排除中枢神经系统出血（脑转移患者需谨慎）
   - 评估凝血功能（PT、APTT、血小板计数）
   - 血小板 < 50,000/μL 时暂停/延迟抗血管生成治疗
3. **出血监测**：
   - 每周期前询问有无咯血、黑便、鼻出血等
   - ≥3级出血 → 永久停用抗血管生成药物
